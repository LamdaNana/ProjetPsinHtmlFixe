# ProjetPsinHtmlFixe
Projet pour apprendre à utiliser du CSS sans toucher à la partie html du code, Mai 2026
Ce projet a été réalisé dans le cadre de ma formation PSIN afin d’apprendre à utiliser le CSS et à améliorer la mise en forme d’un site web à travers la création de styles personnalisés.

L’objectif était de modifier uniquement la partie CSS d’un projet existant, sans toucher au fichier index.html, afin d’améliorer l’apparence d’un formulaire de contact de support informatique.

Le projet consistait à retravailler l’interface graphique du site en modifiant les couleurs, les boutons, les champs du formulaire et la disposition des éléments pour obtenir un rendu plus moderne et agréable visuellement.

Ce travail m’a permis de mieux comprendre le rôle du CSS dans la personnalisation d’une interface web, ainsi que l’importance de la séparation entre la structure HTML et le design du site.
